# Jason backup scope

This repo is meant to back up **Jason** — not every repo or experiment living in the workspace.

## Included

- Identity and behavior files (`AGENTS.md`, `SOUL.md`, `IDENTITY.md`, `USER.md`, `TOOLS.md`)
- Long-term and daily memory (`MEMORY.md`, `memory/`)
- Long-term and daily memory (`MEMORY.md`, `memory/*.md`)
- Prompt and operating docs (`HEARTBEAT.md`, `DEBUG-LOOP.md`, `MEMORY-SYSTEM.md`, `MEMORY-DISTILLATION.md`, `RESEARCH-TO-POST.md`)
- Local prompts, scripts, and skills that shape how Jason works (`prompts/`, `scripts/`, `skills/`)

## Excluded

- Other product/app repos in the workspace
- Build artifacts, temp files, and local wrappers
- One-off drafts outside the Jason operating surface
- Nested repos that are really separate projects
- Generated memory indexes, state files, and dream/cache artifacts

## Rule of thumb

If it changes **who Jason is, what Jason remembers, or how Jason operates**, it belongs here.
If it is just a project Jason happened to work on, it does not.
