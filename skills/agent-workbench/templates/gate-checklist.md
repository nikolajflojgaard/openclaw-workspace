# Gate Checklist

- [ ] The result still matches the user's request.
- [ ] Scope changes are explicit.
- [ ] User or unrelated worktree changes are preserved.
- [ ] One-writer-per-surface was respected.
- [ ] Validation ran, or the exact gap is stated.
- [ ] Secrets and private data were not exposed.
- [ ] Public/external actions were approved.
- [ ] CI/deploy status was checked when relevant.
- [ ] Final report names commits, links, artifacts, or proposals.
