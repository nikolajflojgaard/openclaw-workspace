- Built and published GitHub repo `JuliansProject` for the Arduino Uno motor-control setup, including a localhost UI, Arduino sketch, and setup README.
- Verified Uno on `/dev/cu.usbmodem31201` with motor PWM on pin 3 and sensor on pin 2; manual PWM control works reliably, with startup threshold around PWM 60.
- Improved the motor UI with a live PWM graph and orange process-variable overlay.
